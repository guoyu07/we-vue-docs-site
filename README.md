The website for wv-vue.

## Getting started

Install dependencies:

``` bash
$ git clone https://github.com/tianyong90/we-vue-docs-site.git
$ cd we-vue-docs-site
$ npm install
```

Generate:

``` bash
$ hexo generate
```

Run server:

``` bash
$ hexo server
```
